# Park-IUL Management System

## Description

Academic project developed for the Operating Systems course at ISCTE.

This project simulates a parking management system and was developed in three stages.

## Part 1 - Bash

Implementation of a parking management system using Bash scripts.

Features:
- Vehicle entry and exit registration
- Parking statistics generation
- Cron automation
- Data processing and validation

Technologies:
- Bash
- Linux
- awk
- sed
- grep
- cron

## Part 2 - FIFO Client/Server

Implementation of a client-server architecture using:

- Named Pipes (FIFO)
- Signals
- Process management

Technologies:
- C
- Linux
- IPC

## Part 3 - Advanced IPC

Implementation using System V IPC mechanisms:

- Message Queues
- Shared Memory
- Semaphores
- Multiple server processes

Technologies:
- C
- Linux
- System V IPC
